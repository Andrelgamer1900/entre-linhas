# Entre Linhas - Biblioteca Digital

Este é o projeto oficial do aplicativo **Entre Linhas**, um simulador de biblioteca digital desenvolvido em **Expo (React Native)** com integração à **Google Books API**.

## 🛠️ Tecnologias Utilizadas
- **Framework:** Expo (React Native)
- **Linguagem:** JavaScript
- **API Externa:** Google Books API (Busca de livros em tempo real)
- **Estilização:** StyleSheet (Design focado em e-readers)

## 📖 Funcionalidades Implementadas (Sprint 2)
- Estrutura base do aplicativo mobile.
- Integração com a API do Google Books para busca de títulos.
- Listagem dinâmica de livros com capas e autores.
- Tela de detalhes do livro com descrição completa.
- Interface responsiva e intuitiva.

## ⚙️ Como Rodar o Projeto
1. Clone o repositório.
2. Execute `npm install` para instalar as dependências.
3. Execute `npx expo start` para iniciar o servidor.
4. Utilize o app **Expo Go** no celular para visualizar.
