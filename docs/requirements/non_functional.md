# Requisitos Não Funcionais

## Desempenho
O sistema deve responder às ações do usuário em até 3 segundos.

## Usabilidade
A interface deve ser intuitiva e de fácil navegação.

## Compatibilidade
O aplicativo deve funcionar em dispositivos Android utilizando Expo.

## Segurança
As credenciais dos usuários serão gerenciadas pelo Firebase Authentication.

## Manutenibilidade
O projeto foi estruturado em módulos utilizando TypeScript para facilitar manutenção e escalabilidade.
